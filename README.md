# Ngôn Ngữ Phát Triển Ứng Dụng (Clean)

Repo “sạch” gồm:

- `Back-End/`: Node.js (Express) + SQL + middleware auth
- `Front-End/`: React (Vite) với các trang: Trang chủ, Đăng ký, Đăng nhập

## Yêu cầu

- Node.js LTS (>= 20)
- PostgreSQL (khuyến nghị)

## Chạy Back-End

```bash
cd Back-End
npm install
cp .env.example .env
npm run dev
```

## Chạy Front-End

```bash
cd Front-End
npm install
npm run dev
```
