# Ngôn Ngữ Phát Triển Ứng Dụng (Clean)

Repo gồm:

- `Back-End/`: Node.js (Express) API
- `Front-End/`: React (Vite)

## Yêu cầu

- Node.js LTS (khuyến nghị)

## Chạy Back-End

```bash
cd Back-End
npm install
copy .env.example .env
npm run dev
```

Mặc định API: `http://localhost:4000`

## Chạy Front-End

```bash
cd Front-End
npm install
npm run dev
```

Mặc định web: `http://localhost:5173`
